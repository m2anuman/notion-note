Create Repo on Git using VS Code and Git Bash

  1. Click on the (+) button and click on New Repository.
  2. Write Repository name and Create repository.
  3. Create a new folder on File Explorer using same name as Repo.
  4. Open that folder on VS Code.
  5. Open terminal and select Git Bash.
  6. Type echo "# repo name" >> README.md and press Enter. (A new README.md file will be created.)
  7. Type git init and press Enter.
  8. Type git add . and press Enter.
  9. Type git commit -m "Meaningful Message" and press Enter.
  10. For 1st time you might see some error message here. Sign in to git via pop-up or link and connect your Git with VS Code.
  11. Type git branch -M main and press Enter. ( Branch will change from master to main.)
  12. Type git remote add origin https://github.com/m2anuman/example.git and press Enter.
  13. Type git push -u origin main and press Enter.
  14. Done. Now reload your repo page.
